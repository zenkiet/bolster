Introduction

Bolster - eCommerce Nuxt.js/Vue.js Template

Local Development Environment Setup

In order to kickstart the development of your web app with Bolster, the very first thing is to setup an NUXTJS development environment.

NUXTJS

    Introduction from Nuxt.js official doc
  • Nuxt is a progressive framework based on Vue.js to create modern web applications. It is based on Vue.js official libraries (vue, vue-router and vuex) and powerful development tools (webpack, Babel and PostCSS). Nuxt goal is to make web development powerful and performant with a great developer experience in mind.
  • If you want to more comfortable with NUXTJS please visit this Tutorial

NPM Packages

  • Before proceding you'll need to install npm packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.

Firebase Setup

  • Before start the development server you need to setup the Firebase Database that's why need to go to Firebase Console it will ask you to login with Gmail if you didn't have.
  • Strongly recommended: Please see the below video for creating a Database & get Firebase Database config credentials




  • And after creating a Firebase Database please copy the crendentials inside the firebaseConfig variable & past it into the /plugins/firebase.js file and inside the firebaseConfig variable
  • That's it and Done!

Development Server

  • Run npm run dev for a dev server. Navigate to localhost:3000/. The app will autometically reload if you change any of the source files.

Building

  • Run npm run build to build the project.
  • Run npm run generate to build the production ready deployment project. The build artifacts will be stored in the dist/ directory.

More Help

  • To get more help on the Nuxt.js check out the NUXTJS Guide.

Fonts

By default, the template loads Nunito+Sans font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "index.html" file: /assets/scss/styles/style.scss


<link href="https://fonts.googleapis.com/css?family=Nunito+Sans:200,200i,300,300i,400,400i,600,600i,700,700i,800,800i,900,900i&display=swap" rel="stylesheet">
